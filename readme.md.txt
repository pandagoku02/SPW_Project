Steps - 
Install NPM and Nodejs, MongoDB, MongoDB Compass

1.Unzip both the folders api and client 

2.Open terminal in each folder and write command - npm install

3. After npm install is done write command in both terminal - npm start

4. Wait for the server to start and the default browser will automatically open with the application

and in case it does not, open browser and go to http://localhost:3000

You may start using the application now, with all the functionality
5. For Admin login - signup as a normal user
6. Open MongoDB Compass - select the Daily Post collection - go to users
7. Edit the Role for the user you want to be admin as - "Admin" 
as it is case sensitive
